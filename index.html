Project Idea

Overview

Discord is a popular VoIP and instant messaging software in the video gaming community, and currently has over 250 million users. The application is available on both PC and mobile, and also supports the use of ‘bots’, which are automated accounts with coded functionality, such as moderating chatrooms to delete messages that contain any preset words. Final Fantasy XIV (FFXIV) is a Massive Multiplayer Online (MMO) game, and there is a job in the game that allows you to gather materials at set times within the game’s clock (Eorzea Time/E.T.); these are referred to as Timed Nodes. I would like to create a Discord bot that will notify the user when Timed Nodes will spawn.


Motivation

As an avid gamer, I use Discord as a one-stop platform for almost everything. I use it to communicate with friends both over voice chat and text, and also for sending images and information. There is already a website that will inform you of these Timed Nodes spawning, but I find it quite difficult to navigate, and easily forgotten about due to its inconvenience. Having a bot programmed with the same information on Discord, will prove to be a lot more convenient. I’ll also gain a better understanding of the programming behind a bot, and be able to transfer these skills to other future projects, such as creating a bot that can moderate a community Discord chatroom.


Description

This project will be a Discord bot that will be added to a personal server, and will respond to certain commands with set responses, written in Javascript. This will begin with a bot that says ‘hello!’, then escalate.

Discord is an “all-in-one- voice and text chat for gamers” that is widely used in communities worldwide. Upon creating a Discord account, you’ll receive a unique ID and also be able to create your own ‘server’. Through this server, you can invite other users to join with a URL, where they gain access to text and voice channels. With this unique ID, users can be ‘mentioned’ in the text channels, and it will send a notification to them specifically. Discord bots take on the form of users, and are able to send and moderate messages in text channels, depending on their function.

In FFXIV, Timed Nodes will spawn in set locations within the game’s world whenever the in-game clock ticks to a certain hour. These nodes are only available for set periods of time, often between 5 to 10 minutes in Real Time (R.T.), before they disappear for a set interval, usually around 12 to 24 hours (E.T.) or 35 to 70 minutes (R.T.). The bot will be programmed with information about the Timed Nodes. This will include:

  • A countdown (R.T.) until its next spawn
  • Image of the in-game item
  • X, Y, Z co-ordinate of its in-game location
  • Name of the in-game zone
  • Set spawn times (E.T)

When prompted with certain commands, the bot will respond to the user in the text channel, with the information listed above.

The bot will only respond to commands that are given a set prefix; this will be a symbol followed by a short phrase such as !node or ?node. The bot will not respond unless a set phrase is used. An example of a correct command, with ! as the example would be '!node Tender Dill', whereas ‘!time Tender Dill’ would not have any results. While the prefix is correct, the phrase is not programmed into the bot, therefore there wouldn’t be a response.

When this command is used, all information on the in-game item of Tender Dill will be sent by the bot in response. This will include:

  • A countdown (R.T.) until its next spawn
  • Image of the in-game item
  • X, Y, Z co-ordinate of its in-game location
  • Name of the in-game zone
  • Set spawn times

One of the key functions of the bot, would be a ‘!notify’ command, that will mention a user that uses it, five minutes before a specified node spawns into the world. For example, a user typing ‘!notify Tender Dill’ into a designated chatroom will register/unregister users to a list. Whenever the node spawns, the bot will send a message into a text channel in the server, mentioning any of the users that have signed up for a notification about Tender Dill spawning. Within the message, the following details will also be listed:

  • Image of the in-game item
  • X, Y, Z co-ordinate of its in-game location
  • Name of the in-game zone

Tools and Technologies

  • A computer or phone with Discord installed is a necessity for this project, in order to create the server for the bot and user to join. It is also necessary to view the resulting outputs.
  • A monitor for information to be displayed.
  • A router for internet access. This is neccessary for Discord itself, along with access for Discord's API library along along with Node.js, which is required for the JavaScript.
  • A keyboard and a mouse for user inputs.
  • A private server for the bot to be hosted on.

Skills required

A program will need to be written for the project in Javascript. As I do not have any prior experience, I will seek online resources and help from others well versed in the language, learning how to create a bot that will respond to my inputs. Over time, I will learn how to implement more complicated features, and eventually achieve my project idea. Outcome

If the project is successful, the Discord bot will function as required, and will deliver whatever relevant information is requested to the user about timed nodes, this will mitigate the issue of inconvenience associated with other resources that accomplish the same task. This project will not only impact myself, but it can be shared with other FFXIV players that face the same issues, being a convenient way to check information about nodes without having to open a web browser.
